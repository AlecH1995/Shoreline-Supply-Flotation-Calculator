# Shoreline Supply Flotation Calculator

## Deploy to Vercel (recommended)

1. Upload this folder to a GitHub repository
2. Go to vercel.com and sign in with GitHub
3. Click "Add New Project" and select your repo
4. Click Deploy â€” no configuration needed

Your live URL will be: https://your-project-name.vercel.app

## Embed on your website

Once deployed, add this iframe anywhere in your site HTML:

```html
<iframe 
  src="https://your-vercel-url.vercel.app" 
  width="100%" 
  height="900px" 
  frameborder="0"
  style="border: none;">
</iframe>
```

## Run locally

```bash
npm install
npm start
```
